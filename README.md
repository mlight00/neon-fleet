# 네온 함대 (Neon Fleet)

라스트워 낚시 광고에 나오는 "그 미니게임"을 낚시 없이 온전히 즐길 수 있게 만든 무료 웹게임.
드론 편대를 좌우로 움직여 크리스탈과 워프 게이트로 편대를 불리고, 트랙 끝의 모선을 격파하세요.

- 개인 취미 프로젝트 · 광고 없음 · 과금 없음 · 설치 없음
- 순수 JavaScript + Canvas 2D (빌드 도구 없음)

## 실행 방법

ES 모듈을 쓰기 때문에 파일을 직접 열면 안 되고, 간단한 로컬 서버로 띄웁니다:

```bash
# 프로젝트 폴더에서
python -m http.server 8321
# 브라우저에서 http://localhost:8321 접속
```

폰에서 하려면 같은 와이파이에서 `http://<PC IP>:8321` 로 접속하거나, GitHub Pages에 올리면 됩니다.

## 조작법

| 기기 | 조작 |

| 모바일 | 화면 터치 후 좌우 드래그 |

사격은 자동입니다. 조작은 좌우 이동뿐!

## 게임 요소

- **에너지 크리스탈** (노란 육각형): 쏘면 숫자가 깎이고, 다 깎으면 그 숫자만큼 드론 획득
- **보급 수송선** (청록 육각 컨테이너): 부수면 드론 지급 — 뒤로 갈수록 단단하고 보상이 크다
- **격파 현상금**: 중형(+4)·대형(+12) 크리처를 부수면 드론 회수
- **워프 게이트**: 청록(+, ×)은 이득, 마젠타(−, ÷)는 손해 — 편대 중심이 지나간 쪽이 적용
- **운석**: 닿으면 아프고, 부수면 코인
- **화력 모듈** (금색 별): 10초간 화력 2배

## 함선 진화 (드론 소모형 — 모은 드론이 기함의 재료가 된다)

드론을 모아 진화 비용을 채우면 **모은 드론이 기함에 흡수**되며 1티어 진화한다.
흡수량의 25%만 새 호위대로 재사출되고 나머지는 소멸 — 다음 진화는 처음부터 다시 모은다.

스카웃(시작) → **인터셉터**(60기) → **스트라이커**(140기) → **커리어**(280기) → **드레드노트**(500기) → **타이탄**(800기)

- 총 화력 = 드론 수 + 기함 파워: 바친 드론만큼 기함 파워가 영구 상승해 진화는 항상 저축이다.
- 강등 없음 — 바친 재료는 돌려받지 않는다. 진화 순간 보호막 1회가 생성된다.
- 한 판에 2~3회 "모으고 → 바치고 → 다시 모으는" 사이클이 돌도록 트랙이 길다 (14구간).

## 무기 3종 (라이덴 스타일)

- **발칸** (청록, 확산): 넓게 뿌리는 안정형 — 적 떼에 강함
- **레이저** (하늘, 관통): 정면 고화력 — 대형 크리스탈·보스에 강하지만 조준 필요
- **호밍** (금색, 유도): 절대 안 빗나감 — 회피 중에도 딜 유지, 대신 화력 낮음

무기 선택: 판 시작 직후 **3레인 무기 게이트**, 중반 **보너스 게이트**(+드론/레벨업/실드), 트랙에 떨어지는 **색깔 캡슐**(같은 색=레벨업, 다른 색=교체·레벨 유지).

## 적: 보이드 스웜

- **샤드**(가시 파편) / **리퍼**(낫팔, 지그재그) / **브루드 캐리어**(장갑 육각, 샤드 사출) — 닿으면 크게 아픔
- **저격 드론**: 조준탄 + 두 번에 한 번 3점사 (빨간 링 = 발사 예고)
- **고정 포탑**: 5방향 부채꼴탄 ↔ 8방향 원형탄 번갈아
- **사이드 위버**: 탄 커튼 + 3발마다 조준탄
- 적은 전 구간 **2배 밀도**(미러 배치)로 출현한다
- **중간보스** (스테이지 2+): 직전 스테이지 보스가 트랙 중반에 일반 적처럼 등장해 지나간다. 격파하면 드론 대량 회수, 놓치면 그냥 통과
- **스테이지 보스 5종 로테이션** — 각자 고유 패턴:
  - 하이브 퀸: 산란 폭주 (소환 집중)
  - 리퍼 로드: 참격 부채 7연발 + 빠른 이동
  - 볼텍스 마우: 좌우로 쓸어내는 나선 탄류
  - 옵시디언 클로: 좌우 집게 협공탄 + 단단한 몸(HP 1.2배)
  - 보이드 세라프: 회전 깃털 원형탄

## 폴더 구조

```
index.html          진입 페이지
css/style.css       레이아웃/오버레이 스타일
js/balance.js       모든 밸런스 수치 (여기만 고쳐서 난이도 조정)
js/logic.js         게임 규칙 순수 함수 (테스트 대상)
js/collision.js     충돌 판정 (테스트 대상)
js/chunks.js        트랙 구간 패턴 15종 + 추첨 (테스트 대상)
js/save.js          기록 저장 (테스트 대상)
js/input.js         터치/마우스/키보드 입력
js/entities.js      게임 개체 (편대/적/아이템/보스)
js/render.js        네온 렌더 헬퍼, 스타필드, HUD
js/ui.js            타이틀/결과 화면
js/main.js          게임 루프, 트랙 생성, 상태 관리
tests/              단위 테스트
docs/specs/         게임 설계 문서
docs/plans/         구현 계획
```

## 테스트

node --test tests/*.test.mjs
