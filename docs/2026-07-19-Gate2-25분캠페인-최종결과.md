## 완성된 25분 캠페인 시스템

- **6지역 시간 진행**: COLD WAKE → (섹터2) → FURNACE LINE → BROKEN ARMADA → CHOIR VEIL → CROWN CORE, 보스 B8·B9·B10·B11·B22·B7. 지역 보스는 Gate 1의 검증된 양측 클램프를 지역 TTK 목표(B8~11 30~45·B22 45~60·B7 60~90)로 재사용.
- **§7.3 함체 T0~T5 기능**: 등급마다 이동 반응·공명 증폭·측면 포대(T4)·Apex 화면 지배(T5) 누적. 매 프레임 tier 파생으로 진화/강등과 동기.
- **§7.2 세 번째 슬롯**: 13분(780s)에 자율 전투기 편대 3기 합류. 독립 조준·사격, 별도 피해 집계, HUD 세 번째 슬롯.
- **§7.4 경로 선택**: ~4분마다 2택. 각 옵션이 밀도·수리·무기강화·공명·호위·보호막 중 ≥2축을 실제로 바꿈(상한 무효 시 폴백으로 2축 보존).
- **§7.5 지역 적 구성**: 지역마다 다른 역할 조합(군집·저격·장갑·모선·전격·혼성) + 주기적 정예 웨이브(★ 3.2×HP).
- **전용 결과 패널**: 6지역 보스 TTK 표 + Gate 2 시스템(함체 등급·함대·경로·정예·공명) + 전체 피해 비율.
