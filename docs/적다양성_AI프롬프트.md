## A. 새 적 2종 그림 (나노바나나)

**작업 방법**: 기존 보스 그림(`assets/styleC/B7.png`)을 **먼저 첨부**해 화풍을 통일한 뒤, 아래 프롬프트를 붙여넣어 생성. 결과는 `E:\workspace\claude\neon-fleet\작업중_보스이미지\`에 `charger.png`, `mine.png`로 저장해 주시면 제가 배경 제거 → 게임 규격(`assets/styleC/B12·B13.png`)으로 넣습니다. (배경은 분홍 단색으로 나오는 게 정상)

### 돌진병 (Charger)
```
첨부 이미지와 똑같은 페인티드 카툰 스타일로, 우주 슈팅 게임의 소형 돌진 적을 그려줘.
- 위에서 내려다본 탑다운, 뾰족한 앞부분이 화면 아래를 향한 쐐기/화살촉 형태의 공격형 소형 함선.
- 어두운 보라+청록 금속 갑각, 앞쪽이 날카롭게 각진 램(충각) 구조, 뒤쪽에 추진 노즐 2개.
- 중앙에 붉게 이글거리는 작은 코어 1개.
- 좌우 대칭, 이미지의 80%를 채우는 크기, 잘림 없음.
- 배경은 순수한 마젠타 단색(#FF00FF), 그림자·그라데이션 없이. 몸에는 마젠타/분홍 금지.
- 글자·워터마크 없이 정사각형 1장.
```

### 기뢰 (Mine)
```
첨부 이미지와 똑같은 페인티드 카툰 스타일로, 우주 슈팅 게임의 부유 기뢰를 그려줘.
- 위에서 내려다본 탑다운, 둥근 구체 몸체에 바깥으로 뾰족한 가시(스파이크) 8개가 방사형으로 돋은 해상기뢰 형태.
- 어두운 보라+청록 금속 질감, 표면에 리벳과 경고 줄무늬.
- 중앙에 노란빛으로 깜빡이는 신관 코어 1개(위험 표시).
- 좌우 대칭, 이미지의 75%를 채우는 크기, 잘림 없음.
- 배경은 순수한 마젠타 단색(#FF00FF), 그림자·그라데이션 없이. 몸에는 마젠타/분홍 금지.
- 글자·워터마크 없이 정사각형 1장.
```

> 변이(어픽스) 아이콘은 지금 기호(★ ◈ ☣ ◎ 등)로 잘 읽혀서 **따로 그릴 필요 없습니다.** 굳이 원하시면 위 방식으로 6종 소형 아이콘 시트를 만들 수 있으니 말씀 주세요.

## B. 효과음 (선택) — 텍스트→오디오 AI (예: ElevenLabs SFX)

지금은 기존 폭발/피격음을 재사용합니다. 전용 소리를 원하면 아래 문구로 생성해 `assets/sound/`에 `nf_sfx_<이름>.ogg`로 넣어 주시면 코드에 연결합니다.

- **엘리트 처치** (`elite_die`): `deep powerful sci-fi explosion with a bright metallic ring and a short triumphant chime, punchy, 0.8s`
- **기뢰 무장 경고** (`mine_arm`): `rapid electronic warning beep, rising urgency, retro arcade, 0.5s`
- **돌진 예고→돌진** (`charger_dash`): `sci-fi energy charge-up then a sharp whoosh dash, aggressive, 0.7s`

지정해 주시면 `js/audio.js`에 등록하고 해당 순간(엘리트 사망·기뢰 armed·돌진 시작)에 연결하겠습니다.
