## 캡처 주의

- 모바일 파일은 실제 PNG 390×844다.
- 데스크톱 검증 당시 페이지의 `innerWidth×innerHeight`는 1280×720으로 확인했다.
- Codex 앱의 브라우저 캡처 버퍼는 오른쪽 앱 패널 폭을 제외해 데스크톱 PNG를 1073×720으로 저장했다. 따라서 데스크톱 파일은 독립 검증 참고용이며, Claude가 제출해야 할 **파일 자체 1280×720 증거를 대신하지 않는다**.
- `*-full.png`는 긴 자산 랩 페이지의 전체 문서 진단 캡처다. 판정용 대표 이미지는 위 표의 일반 뷰포트 파일이다.
