# 적 다양성 1단계 — 설계 기록

로그라이크 재개편 로드맵의 **1단계**(적 다양성). 이후 ② 진화 모듈(빌드) → ③ 무한 심연+난이도 등급 → ④ 기함 해금 순으로 진행. 이 단계는 현행 스테이지 구조 위에서 독립적으로 동작하며 **거의 전부 코드**(새 그림 의존 없음).

## 목표
무한히 이어질 원정에서 "같은 적만 계속 나온다"는 반복감을 없앤다. 새 몬스터를 대량으로 그리는 대신, **변이(어픽스)로 같은 적을 다양하게** 만드는 로그라이크 정석을 채택.

## 구성 요소

### 1. 엘리트 변이(어픽스) — `js/affixes.js`
적 스폰 시 확률로 특성 1~2개가 붙는다. 색 오라 + 아이콘으로 표시(새 스프라이트 0장). 스테이지가 깊을수록 자주·중첩된다(`baseChance + chancePerStage`, `twoAffixStage`).

| 변이 | 효과 | 대상 종류 |
|---|---|---|
| 가속 swift | 이동/발사 빠름 | 전부 |
| 보호막 shield | 첫 피격 무효 | 근접·사격형 |
| 분열 split | 죽으면 소형 2기 | 크리처 |
| 독성 toxic | 접촉 피해 ×1.7 | 접촉형 |
| 엘리트 elite | HP·크기 ↑, 처치 시 드론·코인 대박 | 크리처·포탑·돌진병 |
| 자성탄 magnet | 탄이 편대 유도 | 사격형 |

순수 함수 `rollAffixes`/`applyAffixes`로 분리 → 단위 테스트(9종). 순환 참조 회피 위해 분열은 `entity.splits` 플래그만 세팅(각 적의 자체 사망 코드가 처리).

### 2. 새 적 2종 — `js/entities.js` (벡터, 그림 없이 즉시)
- **돌진병 Charger**: 진입 → 편대 조준 예고(붉은 점선) → 급강하. 회피 타이밍 게임.
- **기뢰 Mine**: 느리게 떠다니다 편대가 armRadius 안에 들면 fuse 후 폭발(반경 피해). 쏘아서 미리 터뜨리면 안전+코인.

### 3. 보스 변주 — `Boss.variantLevel`
로스터 2회차(스테이지 6+, `loop=floor((stage-1)/5)`)부터 같은 스프라이트가 강화판으로: 이름 접미사(II·III…), 시작부터 광폭(`variantAlwaysEnrage`), 서명 공격 탄 수 +2/loop, 발사 주기 단축, HP +12%/loop.

### 4. 청크·스폰 배선
- `chunks.js`: 돌진병(스테이지 3+)·기뢰(2+) 신규 청크 6종, 초반 안전구간에서 위협 배제.
- `main.js`: 적 스폰 헬퍼가 스케일 + `maybeAffix` 롤을 각 인스턴스에 적용(2배 복제본도 개별 롤 → 좌우가 다른 변이).

## 밸런스
모든 수치는 `js/balance.js`의 `affix`/`charger`/`mine`/`bossVariant`에 집약.

## 검증
- 단위 테스트 40종 통과(affix 9 + chunks 갱신 포함).
- 헤드리스 시뮬: 스테이지 3~6 실플레이, 런타임 오류 0, 6종 변이·돌진병·기뢰·중간보스·보스변주(하이브 퀸 II) 정상.
- (프리뷰 스크린샷 도구는 이번 세션에서 플레이크 — 로직은 위로 검증됨.)

## 그림/사운드 (선택적 폴리시)
