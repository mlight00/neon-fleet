## 2. 제작 범위

### 기함과 함선

- H0 프로토타입부터 H5 타이탄까지 6단계 기함 계보
- H2/H5는 동일 계보를 유지하면서 크기, 장갑 질량, 코어, 지휘 실루엣이 명확히 달라지도록 재설계
- H2 지휘 프레임: 기함 본체와 드론/순양함이 섞이지 않도록 색상과 크기 위계를 분리
- H2/H5 전체 함선은 흰색·건메탈·금색 지휘 라인 중심으로 통일하고 청록색은 제한적인 에너지 포인트에만 사용

### 무기와 발사체

- 무기 마운트 9종
- 발사체 9종
- 무기별 실루엣, 색, 발사 방식, 궤적 차이를 분명하게 설계
- 강화 단계가 단순 수치 증가로 보이지 않도록 마운트와 발사체 모두 시각적 차이를 제공

### 적과 보스

- 역할이 구분되는 일반 적 6종
- B22 아비터: 본체와 파괴 부위 6개로 분리
- B7 하이브 퀸: 닫힘/열림 상태와 파괴 부위 7개로 분리
- 하이브 퀸은 3마리의 평범한 보스가 아니라 한 개체의 다단계 모듈형 보스로 사용할 수 있게 구성

### 배경·스토리·VFX·브랜딩

- 패럴랙스 배경 4장
- 스토리 컷과 통신 초상화 7장
- 업그레이드/장갑 고정/무기 진화/티어 상승/보스 장갑 파괴 VFX 5장
- 타이틀 로고, 앱 아이콘 등 브랜딩 이미지
- 배경 중앙은 전투 가독성을 위해 의도적으로 어둡게 유지

## 3. 검증 결과

### 이미지 기술 검증

- 마스터 PNG 68개: 디코딩, 크기, 알파 채널, 빈 이미지 여부 검사 통과
- 투명 스프라이트: 투명 모서리와 유효 불투명 영역 검사 통과
- B22 부품 레이어 재조합: 원본과 RGBA 최대 차이 0
- B7 부품 레이어 재조합: 원본과 RGBA 최대 차이 0
- H2 지휘 프레임 공통 캔버스: 533×768
- B22 공통 캔버스: 768×705
- B7 공통 캔버스: 768×582
- WebP 알파 스프라이트: RGB/알파 손실 없음
- 불투명 WebP 이미지: 최소 가시 영역 PSNR 38.01 dB

- 기함 계보: 목표 런타임 크기에서 H0~H5 실루엣과 질량 증가 확인
